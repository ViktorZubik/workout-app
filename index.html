Как обновить Apps Script
1. Открой Apps Script своей таблицы
2. Замени весь код (кнопка ниже скопирует новый)
3. Сохрани и задеплой заново: «Развернуть» → «Управление развёртываниями» → ✏️ → выбери «Новая версия» → Сохранить
4. URL остаётся тот же — ничего менять не нужно
